# Swimming Pools Website

Простой сайт для компании по строительству бассейнов, созданный с использованием HTML, CSS и JavaScript.

## Технологии

- HTML5
- CSS3
- JavaScript (чистый JS)

## Особенности

- Адаптивная вёрстка
- Готовый дизайн, полностью сверстанный вручную
- Интеграция с CMS UMI (дополнительные модули и функционал добавлены через систему управления контентом)
